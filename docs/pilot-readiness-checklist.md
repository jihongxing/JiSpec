# JiSpec 试点就绪检查清单

This is now a short index.

It is a boundary checklist only: it does not promise automatic understanding of old repositories. 不承诺自动理解旧仓库。 Each blocker should surface an owner action and the 下一条本地命令.

Use it alongside `Console governance` and `privacy report` when you need the pilot path.

Use these commands when you need the pilot path:

```bash
npm run pilot:ready
npm run jispec -- doctor pilot
npm run jispec -- north-star acceptance
```

## 核心特性

### 1. 智能依赖管理
- 自动发现切片间的依赖关系
- 支持多种依赖类型（hard、soft、conflict、optional）
- 可视化依赖图（Graphviz DOT 格式）
- 循环依赖检测和报警

### 2. 全面的冲突检测
- 4 种冲突类型：资源、版本、逻辑、时序
- 4 个严重程度级别：low、medium、high、critical
- 自动识别可解决的冲突

### 3. 智能冲突解决
- 多种解决策略，每种策略包含：
  - 置信度评分（0-1）
  - 工作量估算
  - 风险和收益分析
  - 自动应用能力
- 支持自动解决和手动干预

### 4. 精准的影响分析
- 三层影响分析：直接、间接、级联
- 风险因素识别和缓解建议
- 工作量估算（测试、文档、沟通）
- 影响分数（0-100）
- 可操作的建议列表

### 5. 版本管理
- SemVer 版本解析
- 版本冲突自动协商
- 版本兼容性检查
- 版本锁文件生成

## 技术亮点

3. **可扩展架构**

4. **完善的报告**

## 使用场景

### 场景 1: 新功能开发前的依赖分析
```bash
# 分析依赖关系，识别潜在冲突
npm run jispec -- dependency analyze
npm run jispec -- dependency detect-conflicts
```

### 场景 2: 修改现有切片前的影响评估
```bash
# 评估变更影响，制定测试计划
npm run jispec -- dependency impact user-auth-slice --change-type modify
```

### 场景 3: 版本冲突解决
```bash
# 自动解析版本冲突，生成锁文件
npm run jispec -- dependency resolve-versions
```
